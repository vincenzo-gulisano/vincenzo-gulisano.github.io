I had the pleasure of participating to the <a href="http://ebsis.info.uaic.ro/school2016/">EBSIS summer school</a> and give a lecture about the data streaming processing paradigm and its use in Fog architectures. The lecture introduces the data analysis challenges and trade-offs in Fog architectures, overviews the data streaming processing paradigm (with links to SPEs such as Apache Storm, Apache Flink and Apache Spark) and introduces several use cases focusing on distributed analysis, parallel analysis and deterministic processing (among others). You can find the slides <a href="http://www.slideshare.net/VincenzoGulisano/the-data-streaming-paradigm-and-its-use-in-fog-architectures">here</a>.
